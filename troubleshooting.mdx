</Info>

## Masalah Umum & Solusinya

<AccordionGroup>
  <Accordion title="❌ Tidak bisa masuk ke server (Connection Timed Out / Can't Connect)">
    **Penyebab Umum:**

    - Salah IP atau port server.
    - Jaringan internet tidak stabil.
    - Server sedang offline atau maintenance.

    **Solusi:**

    - Pastikan IP server adalah: `play.spielnet.xyz`
    - Gunakan port default:
      - Java: `25565`
      - Bedrock: `19132`
    - Coba restart Minecraft dan koneksi internet kamu.
    - Cek status server di Discord atau tanya di #help.
  </Accordion>
  <Accordion title="⚠️ Error: `Outdated Client` / `Outdated Server`">
    **Penyebab:**

    - Versi Minecraft kamu tidak cocok dengan versi server.

    **Solusi:**

    - Gunakan Minecraft versi **1.20.x - 1.21.x** (Java) atau **terbaru** (Bedrock).
    - Jika kamu pakai launcher seperti TLauncher, pastikan memilih versi 1.20.x - 1.21.x secara manual.
  </Accordion>
  <Accordion title="🔒 Lupa password atau tidak bisa login">
    **Solusi:**

    - Jika kamu lupa password akun Spiel, silakan **hubungi admin** melalui Discord atau Whatsapp.
    - Sertakan bukti bahwa kamu pemilik akun (contoh: UUID atau tanggal pertama login jika ingat).
  </Accordion>
  <Accordion title="🚫 Gagal registrasi (`Already registered`)">
    **Penyebab:**

    - Akun kamu sudah terdaftar sebelumnya.

    **Solusi:**

    - Gunakan perintah `/login <password>` untuk masuk.
    - Jika lupa password, ikuti langkah reset via Discord atau Whatsapp.
  </Accordion>
  <Accordion title="🕹️ Delay / Lag saat bermain">
    **Penyebab:**

    - Ping tinggi karena koneksi internet atau lokasi jauh dari server.

    **Solusi:**

    - Pastikan tidak sedang mengunduh atau streaming saat bermain.
    - Gunakan koneksi Wi-Fi yang stabil.
    - Hindari menggunakan VPN saat bermain (kecuali diperlukan).
  </Accordion>
  <Accordion title="💀 Spawn mati terus / spawn trap / dibunuh terus oleh player lain">
    **Solusi:**

    - Gunakan command `/spawn` atau `/home` untuk keluar dari lokasi berbahaya.
    - Laporkan pemain yang menyalahgunakan teleport atau spawn trap.
  </Accordion>
  <Accordion title="🧨 Tidak bisa hancurkan / buka peti di area tertentu">
    **Penyebab:**

    - Area tersebut telah di-claim oleh pemain lain.

    **Solusi:**

    - Hubungi pemilik land jika ingin bekerja sama.
    - Jangan grief land orang lain – itu melanggar peraturan\!
  </Accordion>
</AccordionGroup>
