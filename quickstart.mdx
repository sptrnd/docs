## Mulai Bermain dalam 3 Langkah Mudah

Panduan singkat ini akan membantumu bergabung ke **Spiel Network** dan langsung menjelajah dunia Minecraft kami!

---

### Step 1: Siapkan Minecraft-mu

<AccordionGroup>
  <Accordion icon="gamepad" title="Versi yang Didukung">
    Spiel Network mendukung **Minecraft Java Edition** dan **Minecraft Bedrock Edition** secara **cross-play**!

    - 🎮 **Java Edition**: Versi 1.20.x (disarankan versi terbaru)
    - 📱 **Bedrock Edition**: Versi terbaru (Android, iOS, Windows 10/11, Xbox, dsb)

    <Tip>Pastikan kamu tidak menggunakan client modifikasi yang dilarang.</Tip>
  </Accordion>

  <Accordion icon="network-wired" title="Alamat Server">
    Untuk bergabung ke Spiel Network, gunakan informasi berikut:

    - 🌐 **Server Address**: `play.spielnet.xyz`
    - 🔧 **Port Java Edition**: `25565` *(default)*
    - 🔧 **Port Bedrock Edition**: `19132` *(default)*

    <Tip>Pastikan port yang digunakan sesuai dengan edisi Minecraft-mu.</Tip>
  </Accordion>
</AccordionGroup>

---

### Step 2: Masuk & Daftar Akun

<AccordionGroup>
  <Accordion icon="right-to-bracket" title="Cara Masuk">
    Setelah berhasil terhubung, kamu akan langsung masuk ke lobi utama.

    Jika ini adalah pertama kalinya kamu masuk:

    - Ketik: `/register <password> <ulang_password>`
    - Contoh: `/register rahasia123 rahasia123`

    Jika sudah punya akun:

    - Ketik: `/login <password>`
    - Contoh: `/login rahasia123`
  </Accordion>

  <Accordion icon="shield" title="Keamanan Akun">
    Gunakan password yang kuat dan **jangan bagikan** kepada siapa pun. Jika mengalami kendala login, kamu bisa hubungi admin melalui Discord resmi Spiel Network.

    <Tip>Admin tidak akan pernah meminta password secara langsung.</Tip>
  </Accordion>
</AccordionGroup>

---

### Step 3: Pilih Dunia & Mulai Petualangan

<Accordion icon="map" title="Tiga Dunia, Tiga Petualangan Berbeda">
  Spiel Network saat ini menyediakan **3 mode utama** yang bisa kamu nikmati:

  - 🧱 **Survival Mix**  
    Mode campuran survival dengan ekonomi, shop player, claim tanah, job, dan fitur interaksi komunitas lainnya.

  - 🧙‍♂️ **Isekai RPG**  
    Mode RPG bertema dunia paralel dengan class system, dungeon, quest, skill unik, dan cerita menarik. Cocok untuk kamu yang suka tantangan dan grinding!

  - 🌾 **Vanilla Survival**  
    Pengalaman Minecraft klasik tanpa modifikasi atau plugin berat. Full vanilla dengan fitur minimalis, cocok untuk pemain yang ingin gameplay original.

  <Tip>Kamu bisa berpindah dunia kapan saja melalui NPC di lobi utama.</Tip>
</Accordion>
